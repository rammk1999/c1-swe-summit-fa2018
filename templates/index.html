Bike Sharing Analysis

This webpage is a submission for the Capital One 2018 Winter Software Engineering Summit. This webpage displays some data visualizations and supporting analysis from a data set representing an eight month period of recorded bike sharing data in the city of Los Angeles. From this data I generated 6 graphs which answer various questions and showcase interesting trends in the data. Additionally, there is a prediction widget, that calculates how long someone will be using a bike, given a time and certain gps coordinates.
